Welcome

To start building iOS applications we need a program called Xcode. Xcode is an Integrated Development Environment (IDE) for developing applications for Apple which is a fancy way of saying that everything we need to make applications for Apple is provided by one program.

Xcode

Just around the time a new iOS version is released to the public, Apple also releases a new version of Xcode. Apple is about to release iOS 10 and Xcode 8.

Xcode 8 beta

We shouldn't just wait for Xcode 8 and iOS 10 to be officially announced to the public. Apple provides Xcode 8 beta so that developers can start anticipating for iOS 10. Xcode 8 beta is where we want to start learning because Xcode 8, Swift 3, and iOS 10 are all going to become the norm very soon.

Download Xcode

We can download Xcode in three steps.

1. Google 'Xcode 8 beta'

2. Login

We can use the same Apple ID that we use to login to iTunes.

3. Download
